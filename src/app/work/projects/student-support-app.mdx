## Overview

A student support application designed to improve academic outcomes and student wellbeing through personalized assistance and resource management.

## Key Features

- **Personalized Dashboard:** Student-specific view of assignments, deadlines, and resources
- **Support Request System:** Streamlined process for seeking academic assistance
- **Resource Library:** Centralized repository of study materials and guides
- **Progress Tracking:** Visual representation of academic progress
