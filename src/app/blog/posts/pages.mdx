## Activer ou désactiver des pages

Le composant `RouteGuard` de Magic Portfolio gère l’affichage conditionnel des pages selon l’objet `routes` du fichier `resources/config.js`.

<CodeBlock
    marginBottom="16"
    highlight="6"
    codes={[
  {
    code:
`const routes = {
    '/':        true,
    '/about':   true,
    '/work':    true,
    '/blog':    true,
    '/gallery': false,
};`,
    language: "tsx",
    label: "src/app/resources/config.js"
  }
]} />

Le code ci-dessus garantit que la page `/gallery` n’est pas accessible et n’apparaît pas dans la navigation.

## Ajouter de nouvelles pages

Lorsque vous créez une nouvelle page, il faut l’ajouter à l’objet `routes` dans le fichier `resources/config.js`.

<CodeBlock
    marginBottom="16"
    highlight="6"
    codes={[
  {
    code:
`const routes = {
    '/':        true,
    '/about':   true,
    '/work':    true,
    '/blog':    true,
    '/gallery': false,
    '/music': true,
};`,
    language: "tsx",
    label: "src/app/resources/config.js"
  }
]} />

Le code ci-dessus garantit que la page `/music` est accessible. Les utilisateurs pourront y accéder après l’avoir ajoutée au menu de navigation dans le fichier `components/Header.tsx`.
