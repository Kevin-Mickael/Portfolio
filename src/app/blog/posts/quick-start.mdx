# Guide de démarrage rapide

Bienvenue dans votre portfolio ! Ce guide vous aidera à démarrer rapidement.

## Premiers pas

1. **Personnalisez votre contenu**
   - Modifiez les informations dans `src/resources/content.js`
   - Ajoutez vos projets dans `src/resources/projects-data.js`

2. **Ajoutez vos images**
   - Placez vos images dans `public/images/`
   - Organisez-les par catégories (projects, gallery, etc.)

3. **Modifiez le style**
   - Personnalisez les couleurs dans `src/resources/once-ui.config.js`
   - Ajustez les styles dans `src/resources/custom.css`

## Structure du projet

```
src/
├── app/           # Pages Next.js
├── components/    # Composants React
├── resources/     # Configuration et contenu
└── utils/         # Utilitaires
```

## Prochaines étapes

- Ajoutez vos projets dans la section Work
- Personnalisez la page About
- Configurez votre formulaire de contact
