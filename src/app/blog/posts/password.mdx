## Ajouter des pages protégées

Pour activer la protection par mot de passe sur des pages spécifiques, ajoutez les chemins des pages à protéger dans l'objet `protectedRoutes` du fichier `resources/config.js`.

Le composant `RouteGuard` gérera automatiquement le contrôle d'accès à ces pages, en demandant une authentification par mot de passe avant d'autoriser l'accès.

<CodeBlock
    marginBottom="16"
    highlight="2"
    codes={[
  {
    code:
`const protectedRoutes = {
  '/work/once-ui': true
};`,
    language: "tsx",
    label: "src/app/resources/config.js"
  }
]} />

Le code ci-dessus garantit que la page `/work/once-ui` ne sera accessible qu'après saisie d'un mot de passe. 

## Définir un mot de passe

Le mot de passe peut être défini dans le fichier `.env.local` pour le développement local et comme [variable d'environnement](https://vercel.com/docs/environment-variables) pour la production.
