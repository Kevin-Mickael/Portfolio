## Style global

Le style de Magic Portfolio repose sur la personnalisation Once UI via les data-attributes. Vous pouvez générer une palette de couleurs personnalisée pour la marque, l'accent et les couleurs neutres sur [Once UI](https://once-ui.com/customize), où vous trouverez les instructions pour l'appliquer.

<CodeBlock
    marginBottom="16"
    codes={[
  {
    code:
`theme:       'dark',         // sombre | clair
neutral:     'gray',         // sable | gris | ardoise
brand:       'blue',         // bleu | indigo | violet | magenta | rose | rouge | orange | jaune | mousse | vert | émeraude | aqua | cyan
accent:      'indigo',       // bleu | indigo | violet | magenta | rose | rouge | orange | jaune | mousse | vert | émeraude | aqua | cyan
solid:       'contrast',     // couleur | contraste
solidStyle:  'flat',         // plat | plastique
border:      'playful',      // arrondi | ludique | conservateur
surface:     'translucent',  // rempli | translucide
transition:  'all',          // tout | micro | macro
scaling:     '100',          // 90 | 95 | 100 | 105 | 110`, 
    language: "tsx",
    label: "src/app/resources/config.js"
  }
]} />

## Effet de fond

Un fond préconfiguré est présent dans `layout.tsx` et peut être modifié dans le fichier de configuration. Définissez les éléments graphiques comme le dégradé, les points, les lignes et la grille, et configurez leur apparence.

<CodeBlock
    codeHeight={24}
    marginBottom="16"
    codes={[
  {
    code:
`const effects = {
    mask: {
      cursor: false,
      x: 50,
      y: 0,
      radius: 100
    },
    gradient: {
      display: true,
      x: 50,
      y: -25,
      width: 100,
      height: 100,
      tilt: 0,
      colorStart: 'accent-background-strong',
      colorEnd: 'static-transparent',
      opacity: 50
    },
    dots: {
      display: true,
      size: 2,
      color: 'brand-on-background-weak',
      opacity: 20
    },
    lines: {
      display: false,
      color: 'neutral-alpha-weak',
      opacity: 100
    },
    grid: {
      display: false,
      color: 'neutral-alpha-weak',
      opacity: 100,
      width: 'var(--static-space-32)',
      height: 'var(--static-space-32)'
    }
}`, 
    language: "tsx",
    label: "src/app/resources/config.js"
  }
]} />
