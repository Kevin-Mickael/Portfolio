## Overview

Developed a sophisticated website for G-Yachts Monaco, a premier yacht brokerage, using Next.js for a seamless and fast user experience. The project includes a custom-built administrative panel powered by Payload CMS, enabling efficient content management and easy site updates. The combination of Next.js and Payload CMS ensures both frontend performance and backend flexibility, providing G-Yachts Monaco with a robust platform for showcasing their offerings.
## Key Features

- **Auto Generating PDF Brochures**: Developed a feature that automatically generates PDF brochures for each yacht listing, providing a convenient way for users to download and share detailed information about the yachts.
- **Integration of Yatco API**: Integrated the Yatco API to fetch real-time yacht listings, ensuring that the website always displays up-to-date information on available yachts.
- **Dynamic content**: Implemented a dynamic content management system that allows the client to easily update and manage the website content, including yacht listings, blog posts, and other information.

## Technologies Used

- **Next.js**: For fast, server-rendered React applications.
- **TailWind CSS**: For building responsive, utility-first designs.
- **React**: For building interactive user interfaces.
- **Payload CMS**: For managing content and data on the website.
- **Yatco API**: For fetching real-time yacht listings.
- **GraphQL**: For querying and retrieving data from the CMS.
